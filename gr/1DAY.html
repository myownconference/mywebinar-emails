Γεια σου __name__,

Αυτή είναι μια υπενθύμιση ότι το διαδικτυακό σεμινάριο, __webinar_name__, είναι προγραμματισμένο να ξεκινήσει στις __webinar_date__.

Σημειώστε ότι η πρόσβαση στην αίθουσα του σεμιναρίου θα ανοίξει __before_start_time__ λεπτά πριν από την έναρξη. Σας προτείνουμε να ελέγξετε την ταχύτητα σύνδεσής σας και να ελέγξετε τις ρυθμίσεις ήχου και εικόνας εκ των προτέρων για να αποφύγετε τεχνικά προβλήματα κατά τη διάρκεια της συνεδρίας. Εάν έχετε οποιεσδήποτε ερωτήσεις, ανατρέξτε στον οδηγό για τους συμμετέχοντες.

Μπορείτε να επιβεβαιώσετε ή να απορρίψετε τη συμμετοχή σας. Εάν απορρίψετε, δεν θα σας στείλουμε περαιτέρω ειδοποιήσεις για το σεμινάριο.

__will_be__ __will_not_be__

Επιπλέον, μπορείτε να επιβεβαιώσετε τη συμμετοχή σας και να λάβετε την ατζέντα του σεμιναρίου κάνοντας κλικ στο παρακάτω κουμπί. Θα σας στείλουμε ξανά τον ίδιο σύνδεσμο για να βεβαιωθούμε ότι δεν θα χάσετε την εκδήλωση.

Συμμετοχή στο διαδικτυακό σεμινάριο

Αυτός είναι ο μοναδικός σας σύνδεσμος για τη συμμετοχή στο διαδικτυακό σεμινάριο. Η κοινοποίηση αυτού του συνδέσμου θα σας αποτρέψει από την πρόσβαση στη συνεδρία. Ο σύνδεσμος είναι αποκλειστικά δικός σας και ανυπομονούμε για τη συμμετοχή σας.

Θα σας στείλουμε μια υπενθύμιση με τον σύνδεσμο του σεμιναρίου μία ώρα πριν ξεκινήσει.

Ανυπομονούμε να σας έχουμε μαζί μας σε αυτήν την εκδήλωση!
